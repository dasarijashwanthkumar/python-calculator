# Python Basics Projects

A collection of beginner-friendly, terminal-based Python scripts designed to practice fundamental programming concepts like loops, conditional logic, functions, and user input handling.

---

## 🚀 Projects Included

### 1. Simple Python Calculator (`calculator.py`)
A terminal-based command-line calculator built with Python that performs basic arithmetic operations.

* **Features:**
  * **Addition:** Adds two floating-point numbers.
  * **Subtraction:** Subtracts the second number from the first.
  * **Multiplication:** Multiplies two numbers together.
  * **Division with Zero-Check:** Safely divides numbers and avoids crashes by checking if the denominator is zero.
* **Tech Stack:**
  * **Language:** Python 3.10+
  * **Concepts Used:** Functions (`def`), user input parsing, Type casting (`float`, `int`), and Structural Pattern Matching (`match-case`).

### 2. Password Generator (`Password_generator.py`)
A script that generates secure, random passwords based on user-defined lengths and character preferences.
* **Key Features:** Uses Python's `random` or `secrets` module, string manipulation, and input validation.

### 3. Number Guessing Game (`number_guessing_game.py`)
An interactive game where the computer selects a random number, and the player tries to guess it with dynamic feedback (too high/too low).
* **Key Features:** Game loops (`while` loops), random number generation, and score/turn tracking.
